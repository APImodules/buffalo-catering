Карточки действий с дробями

Подсказка по действиям с дробями

  • Сложение: a/b + c/d = (a×d + b×c) / (b×d)
  • Вычитание: a/b - c/d = (a×d - b×c) / (b×d)
  • Умножение: a/b × c/d = (a×c)/(b×d)
  • Деление: a/b ÷ c/d = (a×d)/(b×c)

1. 1/2 + 3/4

1/2 + 3/4 → общий знаменатель 4:
1/2 = 2/4
2/4 + 3/4 = 5/4

2. 5/6 - 1/3

5/6 - 1/3 → общий знаменатель 6:
1/3 = 2/6
5/6 - 2/6 = 3/6 = 1/2

3. 2/3 × 3/5

2/3 × 3/5 → числители × числители, знаменатели × знаменатели:
2×3 = 6, 3×5 = 15 → 6/15 = 2/5

4. 7/8 ÷ 1/4

7/8 ÷ 1/4 → умножаем на обратную дробь:
7/8 × 4/1 = 28/8 = 7/2

5. 3/5 + 2/7

3/5 + 2/7 → общий знаменатель 35:
3/5 = 21/35, 2/7 = 10/35
21/35 + 10/35 = 31/35

6. 4/9 × 3/8

4/9 × 3/8 → числители × числители, знаменатели × знаменатели:
4×3 = 12, 9×8 = 72 → 12/72 = 1/6

7. 5/6 ÷ 2/3

5/6 ÷ 2/3 → умножаем на обратную дробь:
5/6 × 3/2 = 15/12 = 5/4

8. 7/10 - 3/5

7/10 - 3/5 → общий знаменатель 10:
3/5 = 6/10
7/10 - 6/10 = 1/10
