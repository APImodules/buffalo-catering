Задача 1: упрощение выражения

Русский вопрос:
Упростите выражение: \[ \frac{x^2 - 9}{x + 3},\quad x \ne -3 \]
English question:
Simplify the expression: \[ \frac{x^2 - 9}{x + 3},\quad x \ne -3 \]

📚 Подробное решение (на русском):

В числителе стоит разность квадратов: \[ x^2 - 9 = x^2 - 3^2 = (x - 3)(x + 3) \]

Подставим это в дробь: \[ \frac{x^2 - 9}{x + 3} = \frac{(x - 3)(x + 3)}{x + 3} \]

При условии \(x \ne -3\) можно сократить \(x + 3\) в числителе и знаменателе: \[ \frac{(x - 3)\cancel{(x + 3)}}{\cancel{x + 3}} = x - 3 \]

Ответ: \(x - 3\).

📚 Detailed solution (in English):

The numerator is a difference of squares: \[ x^2 - 9 = x^2 - 3^2 = (x - 3)(x + 3) \]

Substitute this into the fraction: \[ \frac{x^2 - 9}{x + 3} = \frac{(x - 3)(x + 3)}{x + 3} \]

Since \(x \ne -3\), we can cancel \((x + 3)\) in the numerator and denominator: \[ \frac{(x - 3)\cancel{(x + 3)}}{\cancel{x + 3}} = x - 3 \]

Final answer: \(x - 3\).
